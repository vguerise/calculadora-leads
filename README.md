# Calculadora de Leads Qualificados

Painel de análise de Meta Ads com IA.

## Como usar
1. Acesse a URL do GitHub Pages
2. Faça upload do CSV/XLSX exportado do Meta Ads Manager
3. Selecione as campanhas desejadas
4. A calculadora preenche automaticamente e gera análise completa via IA

## Stack
- Frontend: GitHub Pages (HTML/JS puro)
- Backend: Supabase Edge Function (Deno)
- IA: Claude Sonnet via Anthropic API
